# Pinout Reference

## ESP32 (Receiver)

| Function          | Pin      |
| ----------------- | -------- |
| CE                | GPIO 4   |
| CSN               | GPIO 5   |
| SCK               | GPIO 18  |
| MOSI              | GPIO 23  |
| MISO              | GPIO 19  |
| Motor Left (RC1)  | GPIO 16  |
| Motor Right (RC2) | GPIO 17  |
| 5V Power          | VIN / 5V |
| Ground            | GND      |

---

## Arduino Nano (Transmitter)

| Function   | Pin |
| ---------- | --- |
| CE         | D4  |
| CSN        | A7  |
| SCK        | D13 |
| MOSI       | D11 |
| MISO       | D12 |
| Joystick X | A0  |
| Joystick Y | A1  |
| 5V Power   | 5V  |
| Ground     | GND |

---

## Joystick Module

| Function | Connection |
| -------- | ---------- |
| VCC      | 5V         |
| GND      | GND        |
| VRx      | A0         |
| VRy      | A1         |
